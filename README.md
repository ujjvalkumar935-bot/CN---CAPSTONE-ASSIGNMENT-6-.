Network Topology Simulation (Cisco Packet Tracer)
This project demonstrates a basic network topology created using Cisco Packet Tracer. It includes two LANs connected via routers, enabling communication between devices across networks.

🧩 Topology Overview
2 Routers (Router0, Router1)

2 Switches (Switch0, Switch1)

4 PCs (PC0, PC1, PC2, PC3)

Each LAN connects to a switch, which connects to a router

Routers are interconnected to allow inter-network communication

⚙️ Features
Basic LAN setup

Inter-router connectivity

End-to-end communication between PCs

Simulation mode used for packet flow analysis

🚀 How to Use
Open the .pkt file in Cisco Packet Tracer

Switch to Simulation Mode to observe packet flow

Test connectivity using ping between PCs

📌 Purpose
This project is designed for beginners to understand:

Network device connections

Routing between networks

Packet flow visualization

🛠️ Requirements
Cisco Packet Tracer (any recent version)
